# 系统模型名隐藏、默认参数与文本解析增强报告

更新时间：2026-05-29 02:07:52

## 1. 用户可见模型名隐藏

已从前端页面、接口健康检查、README 和系统报告中移除具体图像模型名称。当前用户侧只显示：

```text
图像生成
图像生成服务
本地图像生成服务
```

健康检查不再返回本地模型路径或具体模型名。

## 2. 输出目录调整

生成图像已改为保存到：

```text
/root/static/generated
```

接口 `/images/{filename}` 会从该目录读取图片。已验证新生成图片保存在该目录。

## 3. 默认生成参数

前端默认参数已从快速联调用的 256x256、1 step 调整为正常展示参数：

```text
宽度：768
高度：512
推理步数：12
构图模式：横幅
细节强度：70
留白强度：45
负面约束：文字、水印、现代建筑、摄影质感、卡通风格
```

已验证默认参数生成成功：

```text
output=/root/static/generated/8e689bb2383b460da1d4b9db6b2901ad.png
elapsed_ms=15917
size=768x512
```

## 4. 新增可调参数

前端新增：

```text
构图模式：自动、横幅、竖幅、方幅、远景、近景
细节强度：0-100
留白强度：0-100
负面约束：自定义不希望出现的内容
```

这些参数会写入最终图像生成提示，不只是前端展示。

## 5. 文本解析与场景图扩充

场景图解析已从纯规则升级为：

```text
优先调用文本语义解析服务输出 JSON
失败时回退规则关键词抽取 + 山水画常识补全
```

已验证 `.env` 中的 API Key 可用。示例输入“明月松间照，清泉石上流。”返回：

```text
method=大语言模型解析 + 山水画常识补全
entities=明月、松、清泉、石
expanded_entities=山间、溪流、岩石、树木、雾气
relations=月光穿过松间、清泉流过石上、松树环绕、石承托泉
```

## 6. 参考风格图像

当前本地图像生成管线是文本条件生成，不支持真正的图像条件输入。因此暂未加入“上传参考风格图像后影响生成”的入口，避免做成只上传但不生效的假功能。

如果后续要支持参考图，应接入以下能力之一：

```text
图像编码器 + 风格适配器
图像提示适配模块
风格图像检索后转文本描述
```

两天内更实际的折中方案是：允许上传参考图用于保存和展示，同时手动填写“风格参考说明”；但这不会等同于真正的参考图约束生成。

## 7. 加速 LoRA 说明

已检查外部加速 LoRA 的方向。成熟加速 LoRA 主要面向旧一代或不同图像生成生态，不能保证与当前本地生成管线兼容。为避免引入不可控依赖和生成质量下降，当前没有接入外部加速 LoRA。
